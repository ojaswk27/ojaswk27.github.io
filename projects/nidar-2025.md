# NIDAR 2025 - Autonomous Multi-Drone SAR System

## Overview
Developed a fully autonomous multi-drone Search And Rescue system for disaster management.

## Motivation
Disaster scenarios require rapid response. This project aimed to leverage autonomous drones to locate survivors faster than traditional methods.

## Approach
- Computer vision model training for object detection
- Post-processing and optimization for edge deployment
- Implemented autonomy architecture for multi-drone coordination
- CAD analysis and fabrication for custom drone design

## Results
- Successfully demonstrated autonomous search patterns
- Achieved real-time object detection on edge devices
- Fabricated and assembled custom drone hardware

## What I Learned
- Computer vision model optimization
- Multi-agent autonomous systems
- CAD design and 3D printing for rapid prototyping
