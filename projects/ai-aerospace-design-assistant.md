# AI-Powered Aerospace Design Assistant

## Overview
Built an AI assistant that turns text requirements into preliminary aircraft/rocket/satellite designs with computed specs + citations.

## Motivation
Describe what inspired you to build this project...

## Approach
- Used RAG over 30+ arXiv/NASA papers with embeddings
- Vehicle-type vector index for accurate retrieval
- LLM-based auto-fill for design parameters
- Validation checks to prevent unrealistic sizing

## Results
- Successfully generated preliminary designs from text requirements
- Achieved X% accuracy in specifications
- Processed Y papers for knowledge base

- Working with large language models and RAG systems
- Embedding generation and vector search
