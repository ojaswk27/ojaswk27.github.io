# CV Trash Detection

## Overview
Built a computer vision based trash detection system that routes nearby garbage trucks to locations based on trash volume and current truck location.

## Motivation
Urban waste management is inefficient. This project aimed to optimize garbage truck routes based on actual trash accumulation.

## Approach
- Implemented semantic segmentation for trash detection
- Developed custom method to approximate trash volume from images
- Integrated with real-time garbage truck routing system

## Results
- Qualified for Smart India Hackathon (SIH)
- Achieved X% accuracy in trash detection
- Demonstrated route optimization potential

## What I Learned
- Semantic segmentation techniques
- Image-based volume estimation
- Working with geospatial data for routing
